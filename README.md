# Caption Genius AI

Generate engaging Instagram captions instantly with AI. Simply input your keywords or themes, and let Caption Genius create the perfect caption for your posts.

This project is a micro-SaaS web tool targeting small business owners, social media marketers, influencers, and individuals who struggle with caption writing.

**Features (Planned):**
* AI-powered caption generation.
* Simple, intuitive user interface.
* One-time purchase for lifetime access.

**To run locally (Developer Mode):**
1. Clone this repository.
2. Open `index.html` in your browser.

_Note: The AI generation and payment gateway are placeholders for the live version._
